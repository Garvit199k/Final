About TypeRescue

TypeRescue is an innovative web application that combines typing practice with an engaging dog rescue game. Our platform helps users improve their typing skills while enjoying a fun and meaningful gaming experience.

Key Features

  • Customizable typing tests
  • Engaging dog rescue game
  • Real-time WPM tracking
  • Gender-based themes
  • Global leaderboards
